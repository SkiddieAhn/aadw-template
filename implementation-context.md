## 프로젝트 한 줄 설명
입력 시나리오 파일을 읽고 규칙 기반으로 검증한 뒤 결과 파일과 점수를 생성하는 검증 실행기

## 주요 모듈
- 입력 파일 읽기: 시나리오 파일을 불러와서 파싱
- 검증 실행: 규칙에 따라 입력값 검사
- 결과 파일 생성: 검증 결과를 파일로 저장
- 점수 계산: 최종 점수 산출

## 주의사항
- 출력 파일 형식은 반드시 spec.md 2절 형식을 따를 것
- 외부 API 사용 없음, 로컬에서만 실행
